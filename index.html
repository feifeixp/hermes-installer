使用方式

🚀 立即开始

安装完成后，在终端中运行以下命令：

# macOS / Linux — 重新加载 Shell 环境变量
source ~/.bashrc   # 或 source ~/.zshrc
# Windows — 重启终端（Terminal）即可
# 启动 Hermes
hermes             # 开始对话！

微信聊天

打开微信，直接向已绑定的账号发送消息，Hermes Agent 会自动响应。

配置文件

API 密钥保存在 ~/.hermes/.env，配置在 ~/.hermes/config.yaml。

常用命令参考
命令 说明
hermes 启动交互式对话
hermes model 选择 LLM 模型和提供商
hermes tools 配置启用的工具
hermes config set 设置单个配置项
hermes gateway 启动消息网关（Telegram / Discord 等）
hermes setup 运行完整配置向导
hermes update 更新到最新版本
hermes doctor 诊断环境问题
hermes claw migrate 从 OpenClaw 迁移
