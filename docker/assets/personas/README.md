## 两种用法

1）每人一个独立身份（一个 Hermes 实例 = 一个人）
   把某个 `SOUL/<中文名>.SOUL.md` 重命名为 `SOUL.md` 放进该实例的 HERMES_HOME：
       cp SOUL/江岚.SOUL.md ~/.hermes/SOUL.md
   Hermes 只从 HERMES_HOME 读取，且永不覆盖已存在的 SOUL.md。

2）一个 Hermes，16 个可切换人格
   把 `config.yaml` 内容合并进 ~/.hermes/config.yaml，然后：
       /personality mazhuoyuan
       /personality hanmo

## 说明
- SOUL.md 只承载「身份 / 语气 / 沟通风格 / 要避免什么」，不含作息、健康、关系等生平字段
  （那些按官方建议属于 AGENTS.md）。
- 每个角色刻意保留的缺点映射为「需要自我警惕」一节，在 Hermes 里表现为需要自我校正的沟通倾向。
